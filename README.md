## Purpose

This repository contains a public-facing landing page for a licensed Texas backflow inspection service. The site is built to be simple, fast, mobile-friendly, and easy to deploy through GitHub Pages.

## Site Features

- Responsive single-page contractor layout
- Sticky navigation
- Mobile call-to-action button
- Service sections
- Credential and license highlight areas
- Direct contact calls-to-action on the live page
- GitHub Pages ready
- No build step required

## Deployment

This site is designed for GitHub Pages.

The included GitHub Actions workflow publishes the root directory to Pages whenever changes are pushed to the `main` branch.

## Domain Notes

The site can be routed through a custom domain after DNS is pointed to GitHub Pages. Add a `CNAME` file when the final domain target is ready.

## Maintenance Notes

Before final production use, review final business wording, domain routing, imagery, compliance language, contact links, Google Business Profile link, analytics, and form integration.
